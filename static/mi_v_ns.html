Supervision Determination in a PU Experiment

This page looks at supervision determination in a positive/unlabelled data environment.

Use this page to determine how many labelled examples are required in a Positive/Unlabelled data experiment, for a given total number of samples, and given probability of avoiding type I and type II errors.

Problem Description

Best guess of proportion of positive examples in the general population

Experiment parameters

Significance level, i.e. the chance of a false positive or Type I error
Power of the test, ie. the chance of NOT getting a false negative, or Type II error
Total number of samples
Level of Mutual Information of Interest, determines the scale of the X-axis of the graph

Corresponding solutions

Use these fields to get a precise solution for either value based on the other.

Choosing the effect size you wish to observe will give number of labelled samples required.

Choosing the number of labelled samples available will give the minimum effect size that can be observed.

Mutual Information. Together with Ns shows a particular solution
