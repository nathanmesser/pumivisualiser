Sample Size determination in a PU Experiment

This page looks at sample size determination in a positive/unlabelled data environment.

Use this page to determine how many examples in total required in a Positive/Unlabelled data experiment, for a given proportion of labelled samples, and given probability of avoiding type I and type II errors.

Problem Description

Best guess of the proportion of positive samples in the population

Proportion of training samples that are labelled

Experiment Parameters

Significance level, i.e. the chance of a false positive or Type I error
Power of the test, ie. the chance of NOT getting a false negative, or Type II error
Level of Mutual Information of Interest, determines the scale of the X-axis of the graph

Corresponding solutions

Use these fields to get a precise solution for either value based on the other.

Choosing the effect size you wish to observe will give number of samples required.

Choosing the number of samples available will give the minimum effect size that can be observed.

Mutual Information. Together with N shows a particular solution
